Description: Aplicación web para consultar farmacias de turno en Mar del Plata
Status: Activa

/* TECH */
Frontend: HTML5, CSS3, JavaScript (Vanilla)
Maps: Leaflet + OpenStreetMap
Geocoding: Nominatim (OpenStreetMap)
Hosting: GitHub Pages
Automatización: Script propio (sin tocar servidores ajenos)

/* DATA */
Source: Colegio de Farmacéuticos de General Pueyrredon (público)
Method: Copia manual + lógica de rotación deducida
Update: Automática (el script sabe qué toca cada día)
